TouchArt is an interactive demonstration that enables users to perceive and explore colors through **vibrotactile feedback during virtual painting**. Using a lightweight haptic glove equipped with a piezoelectric actuator, users receive vibration signals corresponding to different colors while interacting with a virtual canvas.

TouchArt는 **가상 페인팅 환경에서 진동 촉각 피드백을 통해 색을 인지하고 탐색할 수 있도록 하는 인터랙티브 데모 시스템**입니다. 피에조 액추에이터가 장착된 경량 햅틱 장갑을 사용하여 사용자는 가상 캔버스와 상호작용하는 동안 색상에 대응하는 진동 신호를 촉각적으로 경험할 수 있습니다.

This work was presented as a **Hands-On Demonstration at the IEEE World Haptics Conference (WHC) 2025**.

본 연구는 **IEEE World Haptics Conference (WHC) 2025**에서 Hands-On Demonstration으로 발표되었습니다.

## Project Overview

Visual arts traditionally rely almost entirely on vision.  
TouchArt explores how **color information can be translated into tactile sensations**, allowing users to experience colors through touch.

전통적인 시각 예술은 대부분 시각 정보에 의존합니다.  
TouchArt는 **색상 정보를 촉각 신호로 변환하여 사용자가 색을 촉각적으로 경험할 수 있는 가능성**을 탐구합니다.

By combining a vibrotactile glove and virtual painting interface, the system enables users to:

햅틱 장갑과 가상 페인팅 인터페이스를 결합하여 사용자는 다음과 같은 방식으로 상호작용할 수 있습니다.

+ Paint on a virtual canvas  
+ Feel color information through vibrotactile feedback  
+ Explore painted regions through tactile interaction  

+ 가상 캔버스 위에 그림을 그리기  
+ 색상 정보를 진동 촉각 피드백으로 느끼기  
+ 촉각 인터랙션을 통해 그림 영역 탐색하기  

This approach demonstrates how tactile feedback can provide an alternative sensory pathway for interacting with visual content.

이 접근 방식은 촉각 피드백이 **시각 정보와 상호작용하는 새로운 감각적 경로**를 제공할 수 있음을 보여줍니다.

## System Architecture

The system integrates several hardware and software components.

본 시스템은 다양한 **하드웨어 및 소프트웨어 구성 요소**를 통합하여 구현되었습니다.

### Hardware

+ Lightweight haptic glove with a **piezoelectric actuator mounted on the forefinger**  
+ **Leap Motion Controller 2** for hand tracking and interaction with the virtual canvas

+ 검지 손가락에 **피에조 액추에이터가 장착된 경량 햅틱 장갑**  
+ 가상 캔버스와의 상호작용을 위한 **Leap Motion Controller 2 기반 손 추적 시스템**

The glove provides localized vibration feedback, enabling users to perceive tactile signals corresponding to colors.

이 장갑은 국소적인 진동 촉각 피드백을 제공하여 사용자가 색상에 대응하는 촉각 신호를 인지할 수 있도록 합니다.

### Software

+ Interactive virtual painting environment implemented in **Unity**
+ Real-time mapping between color parameters and vibrotactile signals

+ **Unity 기반 인터랙티브 가상 페인팅 환경**
+ 색상 파라미터와 진동 촉각 신호 간 **실시간 매핑 시스템**

Color information is mapped to vibration parameters based on cross-modal correspondence studies.

색상 정보는 **감각 간 대응 관계(cross-modal correspondence)** 연구를 기반으로 진동 신호로 변환됩니다.

Example mappings include:

예를 들어 다음과 같은 매핑이 적용됩니다.

+ **Red → 20 Hz vibration**
+ **Green → 200 Hz vibration**
+ **Intensity variation → perceived color brightness**

+ **Red → 20 Hz 진동**
+ **Green → 200 Hz 진동**
+ **진동 강도 변화 → 색의 밝기 인지**

These mappings allow users to distinguish colors through tactile perception.

이러한 매핑을 통해 사용자는 **촉각 신호만으로 색상을 구분**할 수 있습니다.

## Interaction Modes

### Paint Mode

In Paint Mode, users paint directly on a virtual canvas while receiving **real-time vibrotactile feedback corresponding to the selected color**.

Paint Mode에서는 사용자가 가상 캔버스에 그림을 그리는 동안 **선택된 색상에 대응하는 실시간 촉각 피드백**을 받게 됩니다.

This interaction helps users learn associations between tactile signals and colors.

이 과정을 통해 사용자는 **촉각 신호와 색상 사이의 대응 관계를 학습**할 수 있습니다.

---

### Exploration Mode

After painting, users can explore the canvas through touch.

그림을 완성한 후 사용자는 촉각 인터랙션을 통해 캔버스를 탐색할 수 있습니다.

When the user touches a painted region:

사용자가 그림의 특정 영역을 터치하면

+ The system detects the color at that location  
+ Vibrotactile feedback corresponding to the color is generated  

+ 시스템이 해당 위치의 색상을 감지하고  
+ 그 색상에 대응하는 진동 촉각 피드백을 생성합니다  

Through this interaction, users can **identify and distinguish painted regions using tactile cues**.

이를 통해 사용자는 **촉각 단서를 기반으로 그림의 색상 영역을 구분**할 수 있습니다.

## Research Contribution

TouchArt demonstrates the potential of **tactile color representation** as a form of sensory substitution.

TouchArt는 **색상 정보를 촉각으로 표현하는 감각 대체(sensory substitution) 가능성**을 보여줍니다.

The project contributes to research in:

본 프로젝트는 다음과 같은 연구 분야에 기여합니다.

+ Cross-modal perception between touch and color  
+ Haptic interaction for digital art  
+ Accessible design for non-visual interaction with visual content  

+ 촉각과 색상 간 **감각 간 지각 연구**  
+ 디지털 아트를 위한 **햅틱 인터랙션 연구**  
+ 시각 정보에 대한 **비시각적 접근성 디자인**

This system highlights how vibrotactile feedback can expand interaction beyond purely visual modalities.

이 시스템은 진동 촉각 피드백이 **시각 중심 인터랙션을 넘어서는 새로운 감각적 경험을 제공할 수 있음을 보여줍니다.**
