## Project Overview

This project explores the design of a neurofeedback-based game that integrates brain-computer interface signals with interactive gameplay.

The system monitors attention and relaxation levels through EEG signals and translates these signals into game mechanics.

By combining neurofeedback with gamification, the project aims to create engaging training experiences for cognitive regulation.

---

## 프로젝트 개요

본 프로젝트는 **BCI(Brain Computer Interface) 신호를 활용한 뉴로피드백 게임 시스템**을 설계하는 연구입니다.

사용자의 **집중도와 이완 상태를 EEG 신호를 통해 측정**하고, 이를 게임 플레이 요소로 변환합니다.

뉴로피드백과 게임화(gamification)를 결합하여 **인지 훈련을 재미있게 수행할 수 있는 인터랙티브 경험**을 제공합니다.

## Key Features
